Liner Notes

Bach 用十二平均律穷尽了赋格的可能性。这张专辑试图对形式系统做同样的事—— 不是用 CA 模仿人类音乐，而是让计算过程作为原生声音引擎。

每首曲目都是一个数学对象的声音肖像。没有后期处理，没有人为修饰——你听到的就是计算本身。

github.com/4ier/ca-sound · Python + NumPy · 44.1 kHz / 16-bit
