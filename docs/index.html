Category Theory
函子的结构保持、自然变换的路径无关、单子的不确定性链。范畴论——数学的数学。
1
Functor
两个范畴之间的结构保持映射——C 的旋律在 D 中重生，形态改变但关系不变。
2
Natural Transformation
函子之间的态射——F 到 G 的每个分量平滑变形，自然性方块的两条路径殊途同归。

Concurrency

5

6
Mutex Heartbeats
互斥锁下的独奏轮转——等待者的心跳脉冲，持锁者的丰满声线。

Liner Notes

Bach 用十二平均律穷尽了赋格的可能性。这张专辑试图对形式系统做同样的事—— 不是用 CA 模仿人类音乐，而是让计算过程作为原生声音引擎。

每首曲目都是一个数学对象的声音肖像。没有后期处理，没有人为修饰——你听到的就是计算本身。

github.com/4ier/ca-sound · Python + NumPy · 44.1 kHz / 16-bit
